# Cloud Crossovers

Required job parameters:
- date
- source_1 (ex: GSFC, S6)
- source_2 (ex: GSFC, S6) - will default to pulling source_1 for self crossovers
- df_version (ex: p1, p2) - determines which processing version of daily files to use
